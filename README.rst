=======
Pinkman
=======

To use pinkman, follow the following steps:

1. Create a python virtual environment or use system wide environment

.. code-block::

    $ python3 -m venv venv
    $ source venv/bin/activate


2. Install pinkman package with pip.

.. code-block::

    $ pip install pinkman
